# WealthWingman

Welcome to the Wealth Wingman repository for Financial Forecasting! This project leverages OpenAI's GPT-3.5 alongside our proprietary ML algorithms to make market predictions based on news articles and quantitative data analysis. 

## Introduction

In the world of finance, making informed decisions quickly can mean the difference between success and failure. This project aims to provide a powerful tool for financial professionals, traders, and investors by combining the capabilities of OpenAI's GPT-3.5 with our own cutting-edge machine learning technology.

## Features

- **News-Based Predictions**: We harness the power of GPT-3.5 to analyze news articles and extract insights that inform market predictions.

- **Quantitative Data Analysis**: Our private machine learning algorithm processes and analyzes quantitative data to complement news-based predictions, providing a more comprehensive forecasting model.

## Getting Started

To get started with this project, follow these steps:

1. Clone this repository to your local machine.
2. Install the required dependencies by running pip install -r requirements.txt
3. Configure your API keys and authentication details with a .env file.
4. To use the current GPT model for live predictions, navigate to notebooks/live-gpt-version.ipnyb and run the notebook. Read through the cells, as some indicate where you can include your tickers and dates. 

## Usage

To use this system for financial forecasting, refer to the detailed usage instructions in the project documentation. You can configure the system according to your specific needs and input data sources.
